# Eir Chrome Extension

This extension currently has two modes:

- `Sweden`: production export from `1177 Journalen` into `.eir` files.
- `Finland`: trial integration on `MyKanta` with capture bundles and page-level `.eir` downloads.

## Install

1. Unzip this archive.
2. Open Chrome or another Chromium browser.
3. Go to `chrome://extensions/`.
4. Turn on `Developer mode`.
5. Click `Load unpacked`.
6. Select the `ChromeExtension` folder.

## Sweden

1. Open `https://journalen.1177.se/JournalCategories/JournalOverview`.
2. Log in as usual.
3. Use the floating Eir downloader.
4. Download your data as `.eir`.

## Finland

1. Open `https://kansalainen.kanta.fi/`.
2. Log in as usual.
3. Use the `Kanta Capture Bundle` panel.
4. Use `Download .eir` on logged-in data pages.
5. Use `Export` if you want a larger research bundle for mapping the integration.

# eir-stories

`Eir Stories` is a story-first health-journey platform.

Tagline:
`Share your health journey. Help someone else navigate theirs.`

Subdomain:
`stories.eir.space`

## Repo Layout

- `docs/`: static landing page and GitHub Pages assets
- `PRODUCT-BRIEF.md`
- `SITE-ARCHITECTURE.md`
- `STORY-MODEL.md`
- `MVP-SPEC.md`
- `STORY-COMPOSER.md`
- `PII-REVIEW.md`
- `PUBLIC-STORY-PAGE.md`
- `IMPLEMENTATION-ROADMAP.md`

## Local Preview

```bash
cd eir-stories
npm run dev
```

Then open:

`http://localhost:4173/`

## Hosting

This repo is structured for GitHub Pages from the `docs/` folder.
